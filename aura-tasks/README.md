# Aura Tasks - Frontend

This is the frontend application for Aura Tasks, built with React, Vite, and Tailwind CSS.

## 🚀 Features

- **Beautiful UI**: Modern design using Shadcn/UI and Framer Motion.
- **Responsive**: Fully optimized for mobile and desktop.
- **Theming**: elegant light and dark modes.
- **Fast**: Powered by Vite and TanStack Query.

## 🛠️ Tech Stack

- **React**: Component-based UI.
- **Vite**: Ultra-fast build tool.
- **Tailwind CSS**: Utility-first styling.
- **Framer Motion**: Smooth animations.
- **TanStack Query**: Data fetching and caching.

## 📦 Installation

```bash
npm install
```

## 🏃 Running the App

```bash
npm run dev
```

## 🏗️ Building for Production

```bash
npm run build
```
